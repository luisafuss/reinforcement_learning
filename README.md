# reinforcement_learning
Hausarbeit für das Fach "Wissensbasierte Systeme" zum Thema Reinforcement Learning und einer dazugehörigen Implementierung.

## Geplante Gliederung
1. Einleitung/Motivation/Aufgabenstellung
2. Markov decision processes (MDPs)
    1. Environment, States, actions, rewards
    2. Transition function und reward function
3. Verschiedene Designs
    1. Modell basiert
        1.  Online vs. Batch
    2. Modell frei
        0. Passives lernen vs. aktives lernen
        1. Adaptive dynamic programming (ADP) 
        2. Temporal-difference (TD)
        3. Q-learning
        4. Monte-Carlo-Methode
5. Implementierung
6. Bewertung/Fazit
